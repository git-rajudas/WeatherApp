# 🌦️ Weather Forecast Web Application

A modern and responsive Weather Forecast Web Application built using **React.js**, **Supabase Authentication**, and **Weather API Integration**.
The application provides real-time weather updates, multi-day forecasts, saved cities functionality, and a mobile-friendly user experience.

---

# 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature and weather conditions
* 🌍 Location-based weather detection
* 📅 Multi-day weather forecast
* 🔐 User Authentication using Supabase
* ❤️ Save favorite cities
* 📱 Fully responsive mobile-first design
* ⚡ Fast and optimized React frontend
* 🛡️ Protected routes for authenticated users
* 📲 Progressive Web App (PWA) support
* 🎨 Modern UI using Tailwind CSS

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/git-rajudas/WeatherAPI-Project.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd WeatherAPI-Project
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Setup Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_WEATHER_API_KEY=your_api_key
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

---

# ▶️ Run Development Server

```bash
npm run dev
```
